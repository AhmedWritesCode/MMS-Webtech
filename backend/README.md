# ⚙️ Slim Framework 4 - Quick Setup

## 📥 Installation

```bash
composer create-project slim/slim-skeleton [my-app-name]
```

Replace `[my-app-name]` with your project folder name.

## 🚀 Running the App

### Using PHP Built-in Server

```bash
cd [my-app-name]
composer start
```

### Using Docker

```bash
cd [my-app-name]
docker-compose up -d
```

Open your browser at: `http://localhost:8080`

## ✅ Running Tests

```bash
composer test
```
